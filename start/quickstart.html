如何使用 #

在项目中main.js,导入框架。

js
import Vue from 'vue'
// 1. 引入组件库
import cwUI from '@/uni_modules/cw-ui'

// 2. 注册
Vue.use(cwUI)

提示

uniapp 对于目录uni_modules中的组件，会使用easycom自动按需导入. 页面中可以省去导入组件的代码。
